28 июня 1999 года: Шри Чинмой отметил 14-ю годовщину своих занятий тяжелой атлетикой

28 июня 1999 года, в возрасте 68 лет, Шри Чинмой отметил 14-ю годовщину своих занятий тяжелой атлетикой, установив три новых рекорда. Он поднял 900 фунтов (410 кг) на стандартном тренажере для тренировки икроножных мышц сидя, приподнял 1800 фунтов (820 кг) на стандартном тренажере для тренировки икроножных мышц стоя (такой тренажер есть в любом тренажерном зале) и приподнял с подставки над головой 250 фунтов (114 кг) в каждой руке одновременно!
